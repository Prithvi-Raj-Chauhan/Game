
# Game

An economy and real life game made in python  


## Running the game

Clone the project

```bash
  git clone https://github.com/Prithvi-Raj-Chauhan/Game.git
```

Go to the project directory

```bash
  cd Game/
```

Install dependencies

```bash
  pip install requirements.txt
```

Start the game

```bash
  python main.py
```
